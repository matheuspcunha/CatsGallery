
# CatsGallery
[![IDE](https://img.shields.io/badge/Xcode-11.5-blue.svg)](https://developer.apple.com/xcode/)
[![Language](https://img.shields.io/badge/Swift-5-orange.svg)](https://swift.org)

App to fetch cats images from [_Imgur API_](https://apidocs.imgur.com/?version=latest) and show the images on gallery.<br/>
### Details
Used MVVM architecture and native resources like:   [_UICollectionView_](https://developer.apple.com/documentation/uikit/uicollectionview), [_URLSession_](https://developer.apple.com/documentation/foundation/urlsession), [_JSONDecoder_](https://developer.apple.com/documentation/foundation/jsondecoder) e [_NSCache_](https://developer.apple.com/documentation/foundation/nscache).
